Constraint Validation & Blocked Cells

How constraints translate to visual blocking and drop validation

Constraint Types → Visual Rules

DepartmentConstraint (Capacity)

morningCapacity, eveningCapacity, electiveCapacity
→ Cell shows as full when capacity reached for that shift type.
→ Tooltip: "קיבולת בוקר מלאה (3/3)" (Morning capacity full)

DepartmentConstraint (Date Block)

blockedStartDate → blockedEndDate
→ Cell is fully blocked (grayed out).
→ Tooltip: "מחלקה חסומה בתאריכים אלה" (Dept blocked on these dates)

Holiday

Holiday falls within a week's date range
→ Cell is fully blocked (grayed + holiday icon).
→ Tooltip: "חג פורים — אין שיבוצים" (Purim holiday — no assignments)

IronConstraint

Hard scheduling rules (e.g., 1 group per dept per shift)
→ Not visually blocked, but enforced on drop.
→ Toast: "לא ניתן — קבוצה אחת בלבד למשמרת" (Only 1 group per shift)
