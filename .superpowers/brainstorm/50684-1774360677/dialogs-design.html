Technical Implementation Notes

  • All dialogs use Radix AlertDialog — matching existing OtpDialog pattern
  • Forms use React Hook Form + Zod — schema factories accepting t for i18n
  • Date pickers: Need a date picker library. Recommend adding react-day-picker (lightweight, Radix-compatible) with custom day filtering to enforce Sunday-only starts and Thursday-only ends
  • Excel parsing: xlsx (SheetJS) library for client-side file reading + column validation
  • Drag & drop zone: Native HTML5 drag events for file upload (separate from dnd-kit which handles card dragging)
  • Student list in edit dialog: Inline form row for adding, same Excel import component reused for student list import
